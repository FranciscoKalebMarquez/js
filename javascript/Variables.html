Declaring variables

There are four ways to declare a variable in Javascript
1. automatic(bad practice) similar to how python declare a variable

num1 = 50;
2. using var
var num1 = 50;
3. using let
let num1 = 50;
4. using const
const num1 = 50;

When to Use var, let, or const?

Use let for local variables
Use const for Arrays and objects
var if you need to support old browsers
dont use python style

Scope

let and const have block scope, meaning, you cannot use it outside of the block { } you declared it in. undeclared variables have global scope
